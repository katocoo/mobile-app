##  주요 프로젝트 상세: 풍선 더블클릭 게임 (W11)
기존 프로젝트를 고도화하여 실제 게임 루프와 랭킹 시스템을 도입했습니다.

### 1️ 더블클릭 판정 및 콤보 시스템
200ms 이내의 입력을 감지하여 연속 콤보 점수를 부여하는 핵심 로직을 구현했습니다.



### 2️ 시각적 피드백과 애니메이션
`animateFloatAsState`를 활용하여 풍선을 클릭할 때마다 크기가 변화하는 애니메이션을 추가해 타격감을 높였습니다.

### 3️ 기록 및 랭킹 관리
상위 5개의 점수를 정렬하여 표시하는 랭킹 리스트를 하단에 배치해 게임의 지속성을 확보했습니다.

## 기술적으로 배운 점
- **정밀 입력 이벤트 처리:** Double-Tap 판정 등 사용자 인터랙션 설계
- **상태 기반 UI 업데이트:** State 변화에 따른 실시간 UI 렌더링 최적화
- **애니메이션 활용:** 애니메이션 API를 통한 사용자 경험(UX) 개선
